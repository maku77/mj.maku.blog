並びシャンポンの外し順

戦略
更新: 2019-02-09
作成: 2019-02-09

6p6p7p7p

といった並びシャンポンがある形から、平和に切り替えていきたいときは、最終的な待ちがよくなるような順番で落としていきます。

例えば、上記のような形からは 7p を落として 8p を引けば、6p9p 待ちというよい形が残ります（赤 0p の受けがなくなるという欠点はありますが）。 6p を落として 5p 引きでは、4p7p 待ちというよくない待ちが残ってしまいます。

基本的にはできるだけ端の両面待ちが残るように考慮した順番で落とします。 下記はこのようなケースでの並びシャンポンの基本的な外し方です。

  • 1p1p2p2p → 2p を落として 3p 引き狙い（1p4p 待ち）
  • 2p2p3p3p → 3p を落として 4p 引き狙い（2p5p 待ち）
  • 3p3p4p4p → 3p を落として 2p 引き狙い（1p4p 待ち）
  • 4p4p5p5p → 4p を落として 3p 引き狙い（2p5p 待ち）
  • 5p5p6p6p → 6p を落として 7p 引き狙い（5p8p 待ち）
  • 6p6p7p7p → 7p を落として 8p 引き狙い（6p9p 待ち）
  • 7p7p8p8p → 7p を落として 6p 引き狙い（5p8p 待ち）
  • 8p8p9p9p → 8p を落として 7p 引き狙い（6p9p 待ち）

尖張牌である 3p や 7p に近いものから外していくのがよいということですね。

次のような一つ飛ばしのシャンポンではどうでしょう？

2s2s4s4s

基本的にはよい両面待ちへの変化を残すという同じ考えでよいのですが、ここから 2s を落として 3s 引きを狙うというのはさすがに受けが狭いでしょう。 この場合は 4s を落として、3s と 5s の受けを残すのがよいでしょう。

  • 1s1s3s3s → 3s を落として 2s4s 引き狙い
  • 2s2s4s4s → 4s を落として 3s6s 引き狙い
  • 3s3s5s5s → 3s を落として 2s4s 引き狙い
  • 4s4s6s6s → 3s が多く残っていそうなら 4s 切り、7s が多く残っていそうなら 6s 切り
  • 5s5s7s7s → 7s を落として 6s8s 引き狙い
  • 6s6s8s8s → 6s を落として 5s7s 引き狙い
  • 7s7s9s9s → 7s を落として 6s8s 引き狙い
